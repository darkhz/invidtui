#General usage

To start invidtui, type:

invidtui

and the application will initialize, and load a screen like this:

Then, the following methods can be used to navigate within the application:

  • Menu
  • Keybindings

See the Pages section for more details.

The Invidious instance is selected automatically, and can be changed.

#Command-line options

To launch invidtui with preset options, you can use the command-line options, which you can view either by checking the Command line options section or by invoking:

invidtui --help

which will print a list of options with their brief description.
